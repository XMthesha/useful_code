# Useful Code

这个仓库用于存放日常调试、数据查看、工程辅助用的小代码。

## 当前代码

| 文件名 | 作用 |
|---|---|
| `adc_bin_reader/adc_bin_reader.py` | 读取 ADC `.bin` 数据，按十进制、十六进制或二者同时输出 |
| `adc_bin_reader/adc_128ch_reader.py` | 读取 128bit ADC 数据，并按 `CH4_CH3_CH2_CH1` 拆成 4 个 32bit 通道 |
| `adc_bin_reader/README.md` | `adc_bin_reader.py` 的参数说明和使用示例 |
